# Release Notes - v0.4.0

Released: March 2024

## What's New

### Language Features
- ✅ Complete syntax (variables, functions, control flow)
- ✅ Type system with inference
- ✅ Arrays and records
- ✅ Pipeline operator (`|>`)
- ✅ Mutation operator (`=`)
- ✅ Arrow functions
- ✅ Closures

### Standard Library
- ✅ Math functions (abs, sqrt, pow, etc.)
- ✅ Array methods (map, filter, reduce, sum, mean)
- ✅ Statistics (std, variance, normalize, dot)
- ✅ String methods (upper, lower, trim, split)
- ✅ I/O functions (print)

### Tooling
- ✅ REPL with commands (`:help`, `:env`, `:clear`, `:q`)
- ✅ File runner
- ✅ Type checker (warn mode)

## Breaking Changes

None - This is the initial public release!

## Known Issues

- No syntax highlighting for `.kai` files (use JS highlighter)
- Type checker produces warnings only
- No WASM support yet
